# Dataset Information

**Note:** The original dataset (~2GB) is too large for GitHub.

## Download Instructions

**Source:** Kaggle  
**Link:** [(https://www.kaggle.com/datasets/elemento/nyc-yellow-taxi-trip-data?select=yellow_tripdata_2015-01.csv)]

**File needed:** `yellow_tripdata_2015-01.csv`

### Steps:
1. Download the January 2015 file from the Kaggle link
2. Place it in this `/data` folder on your local machine
3. Open the .pbix file in Power BI Desktop
4. The dashboard will automatically load the data

## Why not included?
GitHub has a 100MB file size limit. The CSV file is approximately 2GB.
